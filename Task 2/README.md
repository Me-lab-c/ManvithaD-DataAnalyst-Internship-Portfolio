# 🍽️ Exploratory Data Analysis (EDA) & Business Intelligence on Restaurant Dataset

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) and Business Intelligence on a restaurant dataset to uncover meaningful insights, trends, and relationships within the data.

The analysis includes:
- Descriptive Statistics
- Univariate Analysis
- Multivariate Analysis
- SQL-based Business Queries
- Dashboard Creation in Excel

---

## 🎯 Objectives
- Understand restaurant data patterns and customer preferences
- Analyze ratings, votes, pricing, and restaurant categories
- Perform SQL queries to answer business-related questions
- Build a static dashboard to visualize important KPIs

---

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab
- Microsoft SQL Server (MSSQL)
- Microsoft Excel

---

## 📂 Dataset Features
The dataset contains:
- Restaurant Name
- Online Ordering Availability
- Table Booking Availability
- Ratings
- Votes
- Approximate Cost for Two People
- Restaurant Type
- Cost Category
- Rating Category

---

## 📊 Exploratory Data Analysis (EDA)

### ✅ Descriptive Statistics
- Calculated average ratings and costs
- Analyzed vote distributions
- Examined categorical data distributions

### ✅ Univariate Analysis
- Histogram for ratings distribution
- Bar charts for:
  - Restaurant types
  - Cost categories
  - Online ordering availability

### ✅ Multivariate Analysis
- Scatter plots for votes vs ratings
- Correlation heatmap
- Relationship analysis between variables

---

## 🗄️ SQL Business Queries
Performed SQL queries to answer important business questions such as:
- Top-rated restaurants
- Most popular restaurants by votes
- Impact of online ordering on ratings
- Cost category analysis
- Restaurant type performance

---

## 📈 Dashboard
Created a static dashboard in Microsoft Excel containing:
- KPI cards
- Bar charts
- Histogram
- Business insights

---

## 🔍 Key Insights
- Most restaurants fall under the medium cost category
- Dining restaurants are the most common
- Online ordering is widely available
- Higher votes generally indicate higher popularity and trust
