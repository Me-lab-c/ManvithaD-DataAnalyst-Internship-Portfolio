# Zomato Business Analysis Dashboard

## Objective
To perform a deep-dive business analysis on Zomato restaurant data by defining key KPIs, analyzing restaurant trends, and building an interactive dashboard.

## Tools Used
- Power BI
- SQL
- Excel

## KPIs
- Total Restaurants
- Average Rating
- Average Cost for 2
- Online Order Percentage
- Table Booking Percentage
- Total Votes

## Deep-Dive Analysis
The project includes segmentation analysis based on:
- Restaurant ratings
- Cost categories
- Online ordering
- Customer engagement

## Dashboard Features
- Interactive slicers
- KPI cards
- Scatter plot analysis
- Restaurant type analysis
- Online order analysis
- Table booking analysis

## Key Insights
- Mid-range restaurants dominate the platform.
- Restaurants with online delivery receive higher engagement.
- Premium restaurants attract more customer votes.
- Table booking services are limited among restaurants.
