# Restaurant Data Analysis & Business Insights

## Project Overview
This project focuses on analyzing restaurant data to identify customer preferences, pricing patterns, online ordering trends, and operational insights that can support better business decision-making.

The analysis combines Exploratory Data Analysis (EDA), SQL-based business insights, dashboard visualization, and statistical hypothesis testing to create a complete business intelligence solution.

---

## Objectives
- Analyze restaurant categories and customer preferences
- Identify rating and pricing trends
- Compare online ordering and table booking adoption
- Build an interactive dashboard for business insights
- Validate findings using statistical hypothesis testing

---

## Dataset Information
- Total Records: 148 Restaurants
- Key Features:
  - Restaurant Type
  - Ratings
  - Cost
  - Online Ordering
  - Table Booking

## Exploratory Data Analysis (EDA)
Performed detailed EDA to identify:
- Restaurant category distribution
- Ratings distribution
- Cost distribution
- Online ordering trends
- Table booking patterns

### Key Insights
- Dining restaurants dominate the dataset
- Most restaurants fall under the low-cost category
- Customer ratings are concentrated around the “Good” range
- Online ordering is significantly more preferred than table booking

## Hypothesis Testing
### Business Hypothesis
**H₀:** Online ordering and table booking are equally preferred.

**H₁:** Online ordering is more preferred than table booking.

### Statistical Test Used
- Chi-Square Test

### Result
- p-value = 0.0001
- Null Hypothesis Rejected

### Business Conclusion
The statistical analysis confirmed that online ordering is significantly more preferred than table booking among restaurants.
